## 字段说明

| 字段 | 中文名 | 类型 | 说明 | 取值 |
|---|---|---|---|---|
| event_type | 事件类型 | enum | 区分指令和录制 | command / record |
| created_at | 事件时间 | timestamp | 事件发生时间 | ISO 8601 |
| session_id | 会话ID | string | 会话唯一标识 | — |
| user_id | 用户ID | string | 用户唯一标识 | — |
| source_type | 视频接入方式 | enum | 用户如何接入视频 | upload / stream / camera |
| command_id | 指令ID | string | 指令唯一标识 | —（record 行为空）|
| command_source | 指令来源 | enum | 指令从哪来的 | preset / spark / free_input（record 行为空）|
| scene_category | 场景类别 | enum | 所属场景 | change_clothes / style / face_edit / material / scene / null |
| preset_option | 预设选项 | string | 具体点了哪个按钮 | 皮夹克 / 赛博朋克 / … / null |
| prefill_text | 预填内容 | string | 填入输入框的原始 prompt | — / null |
| command_text | 实际发送内容 | string | 用户最终发送的指令 | — |
| render_status | 渲染结果 | enum | 模型渲染是否成功 | success / fail / timeout |
| latency_ms | 端到端延迟 | int | 发指令到出画面的总耗时（毫秒） | — |
| network_latency_ms | 网络延迟 | int | 客户端到服务端的网络传输耗时（毫秒） | — |
| model_version | 模型版本 | string | 当前模型版本号 | — |
| vote | 用户评价 | int | 点赞点踩 | 1（赞）/ -1（踩）/ 0（未评价）|
| has_download | 是否下载 | bool | upload 模式下是否下载了编辑结果 | true / false / —（非 upload 时不适用）|
| record_start_at | 录制开始时间 | timestamp | 录制片段的起始时间 | —（command 行为空）|
| record_end_at | 录制结束时间 | timestamp | 录制片段的结束时间 | —（command 行为空）|
| record_duration_sec | 录制时长 | int | 录制片段的秒数 | —（command 行为空）|

## 用户行为占比分布（饼图）

### 饼图1：用户反馈行为分布

> 维度："收到渲染结果后，用户做了什么"

| 切片 | 取数逻辑 |
|---|---|
| 点赞 | `vote = 1` |
| 点踩 | `vote = -1` |
| 保存结果 | `has_download = true` 或该 session 有 `event_type = record` |
| 无操作 | `vote = 0` 且未下载且未录制 |

### 饼图2：指令来源分布

> 维度："用户的指令从哪来的"

| 切片 | 取数逻辑 |
|---|---|
| 预设选项 | `command_source = preset` |
| 随机灵感 | `command_source = spark` |
| 自由输入 | `command_source = free_input` |

## 关键分析口径

- **预设 prompt 改写率**：`prefill_text ≠ command_text` 的比例 — 越高说明预设 prompt 写得不够好
- **各场景好评率**：按 `scene_category` 聚合 `vote=1` 的占比
- **预设选项热度**：按 `preset_option` 统计点击次数排行
- **随机灵感转化率**：spark 点击次数 vs `command_source=spark` 的实际发送条数
- **自由输入高频词**：`command_source=free_input` 的 `command_text` 做词频分析
- **模型推理时间**：`latency_ms - network_latency_ms × 2`（去掉往返网络开销）
- **录制深度**：平均录制时长、单会话录制次数分布
