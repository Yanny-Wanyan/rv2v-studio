# RV2V Studio — 流式视频编辑控制台

## 项目背景

实验室上线了一个流式视频编辑模型（RV2V），支持边交互边编辑修改视频内容。本项目是为该模型设计的前端操作展示界面，用于 demo 演示和内部使用。

核心场景：用户接入一段视频（上传/直播流/视频通话），通过文字+图片混合指令实时编辑视频画面（换装、风格转换、背景替换、物体增删等）。

## 页面设计基调

### 视觉风格
- **简约高级，不要暗黑调**。极简，不要把所有功能都堆砌到页面上
- 页面背景：缓慢流动的淡蓝紫渐变（不是纯白），有呼吸感
- 所有卡片/面板：毛玻璃质感（backdrop-filter:blur + 半透明白底）
- 视频空态：深蓝紫渐变（不是纯黑），图标带呼吸动画
- 边框用半透明（rgba），不要实色硬线
- 圆角 14px，阴影柔和

### 布局结构
- **100vh 锁定，不滚动**
- Header（48px）顶部固定
- 视频区：两个 16:9 的视频窗口并排（Camera Feed + Edited Stream），`margin:auto 0` 垂直居中
- 指令栏：贴底，`flex-shrink:0`，高度由内容决定
- 视频和指令栏的位置、大小**全程不动**

### 场景面板交互（方案 A 浮层）
- 场景面板用 `position:absolute; bottom:100%` 从指令栏上方浮出
- 面板底色用实色 `#F0F1F4`（冷灰），和指令栏白底区分
- 点击面板外区域自动关闭
- 视频区和指令栏纹丝不动

## 功能需求与约束

### 视频输入（Camera Feed）
- 三种上传格式：离线视频上传、直播流拉取（RTMP/WebRTC/HLS）、双向视频拨打
- 三种模式在面板 header 中通过按钮切换，对应的输入界面在视频区内展示

### 指令输入区（自上而下）
1. **历史 prompt 区**：始终可见。空态显示引导文字"发送的指令会出现在这里"。有内容后显示历史条目（单行高度，可滚动）。带图的条目有缩略图可点击预览
2. **输入胶囊**：
   - 左侧：图片占位符（3:4 比例，rotate(-6deg) 微倾斜），空态虚线+图标，有图后显示缩略图+删除按钮。一次只能一张
   - 四芒星按钮：随机灵感。未选场景时从通用 prompt 池随机；选了场景从该场景池随机
   - 文本输入
   - 发送按钮（圆形蓝色）
3. **场景标签行**：换装试穿 / 风格转换 / 背景替换 / 物体增删
4. **场景面板**（浮层）：所有场景都用图片卡片网格（不是纯文字标签）。点击卡片 → 文字填入输入框 + 图片填入占位符。**所有选项单选**

### 已移除的功能
- 速度控制（不需要）
- 环境调整滑块（改为"背景替换"图片卡片）
- 底部时序条/波形图（不需要）
- 实时遥测数据/日志（不需要）
- "就绪"状态标识（已替换为新建会话按钮）

### Header
- 左侧：品牌名 RV2V Studio + 蓝色圆点
- 右侧：新建会话按钮（聊天气泡+加号 SVG），hover 有"新建会话" tooltip
- 点击清空所有状态（视频、prompt 历史、图片、面板）

### 通用 UI 规范
- 所有图标按钮 hover 时显示 tooltip（CSS `::after` 实现）
- placeholder 引导词：「接入一段视频，试试替换、增减画面里的任何东西」
- 输入框支持图片+文字混合指令
- Enter 发送

## 技术实现
- 单文件 HTML（含内联 CSS + JS）
- Font Awesome 6.5 图标库（CDN）
- 无框架依赖，纯 vanilla JS
- 商品/场景图片使用 Unsplash CDN
