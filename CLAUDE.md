# RV2V Studio — 流式视频编辑控制台

## 项目背景

实验室上线了一个流式视频编辑模型（RV2V），支持边交互边编辑修改视频内容。本项目是为该模型设计的前端操作展示界面，用于 demo 演示和内部使用。

核心场景：用户接入一段视频（上传/直播流/摄像头），通过文字指令实时编辑视频画面（换装、风格转换、外貌改造、质感变身、背景替换等）。

## 页面设计基调

### 视觉风格
- **简约高级，不要暗黑调**。极简，不要把所有功能都堆砌到页面上
- 页面背景：暖灰 `#F5F3F0`（不是纯白，不是渐变）
- 所有卡片/面板：纯白底 `rgba(255,255,255,.92)` + 柔和阴影
- 视频空态：深蓝紫渐变（不是纯黑）
- 边框用半透明（rgba），不要实色硬线
- 圆角 14px，阴影柔和
- 配色系统：珊瑚橘 `#FF6B4A`（主色）+ 柔紫 `#8B7AFF`（仅光带记忆点）
- 字体：Space Grotesk（标题）+ DM Sans（正文）

### 布局结构
- **100vh 锁定，不滚动**
- Header（48px）顶部固定
- 视频区：两个 16:9 的视频窗口并排（原始画面 + 编辑结果），`margin:auto 0` 垂直居中
- 指令栏：贴底，`flex-shrink:0`，高度由内容决定
- 视频和指令栏的位置、大小**全程不动**

### 场景面板交互（浮层）
- 场景面板用 `position:absolute; bottom:100%` 从指令栏上方浮出
- 面板底色纯白 `#fff`，柔和阴影
- 点击面板外区域自动关闭
- 视频区和指令栏纹丝不动

## 功能需求与约束

### 视频输入（原始画面）
- 两种输入模式：直播流拉取（RTMP/WebRTC/HLS）、摄像头
- 两种模式在面板 header 中通过 segmented control 切换（圆角胶囊分段切换，白底高亮+阴影）
- 播放控制：暂停/继续、退出
  - 仅摄像头模式可暂停，直播流模式禁用暂停
- 音量控制：右下角可展开滑块

### 编辑结果面板
- 清除按钮：清除当前编辑效果，hover 提示"清除当前编辑效果"
  - 有编辑效果时启用（珊瑚橘色），无效果时禁用灰色
- 录制按钮：直播流/摄像头模式均显示录制（无下载分支）
- 渲染状态 badge：待机 / 排队中 / 渲染中 / 已完成 / 失败
  - 排队中：后端返回排队状态时显示（如"排队中，前面 N 人"），等后端接口定义后实现
  - 渲染中时显示等待提示
- 渲染完成时顶部紫色光带闪烁（magic flash）
- 效果时长提示（fxNote）：编辑效果激活时在指令栏上方显示"编辑效果持续约 60 秒后自动消失，可随时点击「清除」手动退出"

### Session 定义
- 同一个视频源 = 同一个 session，更换视频源 = 切换 session
- 切换 session 时清空历史指令展示
- 右上角"新建会话" = 清空当前页面所有状态（类比刷新），不支持查看历史 session

### 并发与排队（待后端接口）
- 常态 4 并发，GPU 满时后端返回排队状态
- 前端用 mini-toast 显示"当前使用人数较多，排队中（第 N 位）"
- 后端分配到资源后通知前端，自动切到渲染中状态
- 原型阶段暂不模拟，等后端接口格式确定后实现

### 指令输入区（自上而下）
1. **历史 prompt 区**：始终可见。空态显示引导文字"发送的指令会出现在这里"。有内容后显示历史条目（单行高度，可滚动）。带图的条目有缩略图可点击预览
2. **输入胶囊**：
   - 左侧：图片占位符（3:4 比例，rotate(-6deg) 微倾斜），当前版本不支持上传参考图，hover 显示 cursor:not-allowed + 提示
   - 四芒星按钮：随机灵感。从通用 prompt 池随机填入输入框
   - 文本输入
   - 发送按钮（圆形蓝色）
3. **场景标签行**：实时换装 / 风格演绎 / 外貌改造 / 质感变身 / 场景切换
4. **场景面板**（浮层）：所有场景都用文字标签。点击标签 → 对应 prompt 填入输入框。**所有选项单选**。换装（需人物主体）、外貌改造（需人脸）、质感变身（把自己变成任何材质）、场景切换（替换背景）面板有提示文案。

### 已移除的功能
- 视频上传模式（不再支持离线视频上传，仅支持直播流和摄像头）
- 重播按钮（无上传模式后不需要）
- 下载编辑结果（统一为录制）
- 速度控制（不需要）
- 环境调整滑块（不需要）
- 底部时序条/波形图（不需要）
- 实时遥测数据/日志（不需要）
- "就绪"状态标识（已替换为新建会话按钮）
- 图片上传参考图（当前版本不支持）

### Header
- 左侧：品牌名 JoyAI-Video-Edit + 珊瑚橘圆点 + 副标题"流式视频编辑体验"
- 右侧：新建会话按钮（聊天气泡+加号 SVG），hover 有"新建会话" tooltip
- 点击清空所有状态（视频、prompt 历史、图片、面板）

### 通用 UI 规范
- 所有图标按钮 hover 时显示 tooltip（CSS `::after` 实现）
- placeholder 引导词：「接入一段视频，试试替换、增减画面里的任何东西」
- 输入框支持文字指令
- 历史指令支持点赞/点踩反馈
- Enter 发送

## 技术实现
- 单文件 HTML（含内联 CSS + JS）
- Font Awesome 6.5 图标库（CDN）
- 无框架依赖，纯 vanilla JS
- 商品/场景图片使用 Unsplash CDN
