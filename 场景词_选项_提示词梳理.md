# 场景词、选项与预设提示词梳理（最新）

## 实时换装（`change_clothes`）

> 面板提示：请确保画面中有清晰的人物主体

| 一级选项 | 对应提示词 |
|---|---|
| 白T恤 | 换上白色圆领T恤 |
| 牛仔夹克 | 换上蓝色牛仔夹克，敞开穿 |
| 黑色西装 | 换上黑色西装，白衬衫打底 |
| 碎花连衣裙 | 换上碎花连衣裙，裙摆随动作飘动 |
| 皮夹克 | 换上黑色皮夹克，拉链半开 |
| 灰色卫衣 | 换上灰色连帽卫衣，宽松版型 |

## 风格演绎（`style`）

| 一级选项 | 对应提示词 |
|---|---|
| 日系动漫 | 画面转为日系动漫风，线条清晰色彩鲜艳 |
| 水墨国风 | 画面变成黑白水墨画风格，留白大量 |
| 赛博朋克 | 画面转为赛博朋克风，霓虹灯光暗色调 |
| 油画质感 | 画面变成油画风格，厚重色彩笔触感 |
| 胶片复古 | 画面加胶片效果，颗粒感和褪色暖调 |
| 像素游戏 | 画面变成像素风，8-bit复古游戏画面 |

## 四芒星随机灵感池（`PROMPTS`）

点击四芒星，从以下 7 条中随机填入输入框：

- 画面转为赛博朋克风，霓虹灯光暗色调
- 背景换为海边日落，沙滩海浪橙红天空
- 画面变成黑白水墨画风格，留白大量
- 换上黑色皮夹克，拉链半开
- 头发变成银白色
- 画面变成像素风，8-bit复古游戏画面
- 肩头趴一只橘猫在打盹
